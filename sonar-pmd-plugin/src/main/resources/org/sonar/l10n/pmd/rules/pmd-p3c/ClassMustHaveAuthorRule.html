Every class should include information of author(s) and date.
Note: When setting template in IDE, the format of author in IDEA is ${USER}, while in Eclipse is ${user} (Case sensitive). The format of date is 'yyyy/MM/dd'.

Examples:

    /**
     * Demo class
     *
     * @author keriezhang
     * @date 2016/10/31
     */
    public class CodeNoteDemo {
    }
